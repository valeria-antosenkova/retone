# Retone — Installation & First-Time Launch

Target platform: macOS, Linux, Windows (PowerShell/CMD). Default shell examples use `zsh`.

---

## How to run
1. Install Node.js (LTS) via installer (https://nodejs.org/en/download)
2. Download the repository
3. In a terminal, navigate to your repository root (`/path/to/retone`) using `cd` command: 
4. Install dependencies: `npm install` 
5. Start dev server: `npm run dev`
6. Open the URL printed in a terminal (something like this: http://localhost:5173)

---

## Prerequisites
- Node.js (LTS) — Node 18+ recommended. Node 16+ can work but v18+ is preferred.
- npm (comes with Node)
- Git (to clone the repo but can be done manually by downloading it)
